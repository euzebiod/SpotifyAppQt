euzebiod/SpotifyAppCpp

Cliente usando a Web Api do Spotify, escrito em C++ e usando o framework qt.

Este cliente acessa os serviços do Spotify usando sua Web Api, que usa o padrão REST.

O sistema se loga no Spotify permitindo o usuário listar sua playlist, buscar músicas através de palavras chaves, adicionar ou excluir músicas de sua playlist, salvar e ler localmente uma playlist e tocar uma musica de uma playlist.
